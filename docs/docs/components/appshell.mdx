# AppShell

The `AppShell` component provides a complete, responsive viewport-bound application frame using **CSS Grid**.

It offers named layout regions (slots) for your brand/logo, navigation, user profile, sidebar, footer, and the main content area — making it ideal for dashboards, admin consoles, and single-page web applications.

## Attributes

| Attribute | Target | Description |
|---|---|---|
| `data-appshell` | Root `<div>` | Activates the CSS Grid application shell |

---

## Slots

| Slot | Required | Description |
|---|---|---|
| `<header>` | No | Top bar — use `data-flex` for brand · nav · profile layout |
| `<aside>` | No | Left sidebar — use `data-stack="xs"` for vertical nav links |
| `<main>` | **Yes** | Primary scrollable content area |
| `<footer>` | No | Bottom bar spanning full width |

---

## CSS Variables

| CSS Variable | Default | Description |
|---|---|---|
| `--ix-appshell-sidebar-width` | `16rem` | Sidebar width on desktop screens (≥ 769px) |

---

## Responsive Behavior

On screens **≤ 768px** the layout automatically switches to a single column — the sidebar stacks below the header. No configuration needed.
