# Box

`Box` is a flexible layout **container primitive**. Configure padding, backgrounds, border radius, borders, and shadows using data attributes.

Use it for cards, info panels, alert blocks, hero areas, and any styled wrapper.

## Attributes

| Attribute | Values | Description |
|---|---|---|
| `data-box` | (present) | Activates the box container |
| `data-padding` | `xs · sm · md · lg · xl` | Internal padding size |
| `data-bg` | `surface · raised · primary · danger · warning · success · neutral · gradient` | Background fill |
| `data-radius` | `none · sm · md · lg · full` | Border corner rounding |
| `data-shadow` | `none · sm · md · lg` | Box shadow depth |
| `data-border` | (present) | Adds 1px solid border |
