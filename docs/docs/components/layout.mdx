# Layout

Layout utility components for structuring page sections, content regions, and UI composition patterns.

All layout components follow the same zero-class, data-attribute design as the rest of ignix-lite. No JavaScript. No wrappers. Pure CSS.

---

## Attributes

### Stack

| Attribute | Type | Values | Description |
|---|---|---|---|
| `data-stack` | `string` | `xs \| sm \| md \| lg \| xl` | Vertical gap between stacked children |

### Flex

| Attribute | Type | Values | Description |
|---|---|---|---|
| `data-flex` | `boolean` | — | Activates horizontal flex row |
| `data-gap` | `string` | `xs \| sm \| md \| lg \| xl` | Gap between flex children |
| `data-align` | `string` | `start \| center \| end \| stretch \| baseline` | Cross-axis alignment |
| `data-justify` | `string` | `start \| center \| end \| between \| around \| evenly` | Main-axis justification |
| `data-wrap` | `string` | `wrap \| nowrap` | Whether children wrap |
| `data-grow` | `boolean` | — | On child: makes it fill remaining space |
| `data-shrink` | `boolean` | — | On child: prevents it from growing |

### Center

| Attribute | Type | Values | Description |
|---|---|---|---|
| `data-center` | `string` | `sm \| md \| lg \| xl \| full` | Max-width: 640px / 960px / 1200px / 1440px / 100% |

### Sidebar

| Attribute | Type | Values | Description |
|---|---|---|---|
| `data-sidebar` | `boolean` | — | Activates sidebar layout |
| `data-sidebar-width` | `string` | `sm \| md \| lg` | Sidebar width: 160px / 256px / 352px |
| `data-sidebar-side` | `string` | `left \| right` | Which side the sidebar is on |

### Cluster

| Attribute | Type | Values | Description |
|---|---|---|---|
| `data-cluster` | `string` | `xs \| sm \| md \| lg \| xl` | Gap between clustered items |

### Switcher

| Attribute | Type | Values | Description |
|---|---|---|---|
| `data-switcher` | `boolean` | — | Activates switcher layout |
| `data-threshold` | `string` | `sm \| md \| lg` | Min child width before stacking: 192px / 320px / 480px |
| `data-gap` | `string` | `xs \| sm \| md \| lg \| xl` | Gap between children |

### Cover

| Attribute | Type | Values | Description |
|---|---|---|---|
| `data-cover` | `string` | `screen \| contained` | `screen`=full viewport height, `contained`=fills parent |
| `data-padding` | `string` | `sm \| md \| lg \| xl` | Inner padding of the cover |
