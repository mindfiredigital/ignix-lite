# Progress

The Progress component provides a lightweight and accessible way to indicate task completion using the native HTML `progress` element.

Ignix Lite progress bars support empty, partial, and completed states.

## Attributes

| Attribute | Type | Description |
|---|---|---|
| `value` | `number` | Current progress value |
| `max` | `number` | Maximum progress value |
