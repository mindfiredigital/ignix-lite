# Section

The `Section` component is a versatile layout primitive that organizes page content into distinct **horizontal blocks**.

It provides vertical spacing (margins), inner padding, background shading, and max-width constraints — giving your pages consistent visual rhythm and structure.

---

## Content Rhythm & Spacing

Use `data-spacing` to control how much vertical whitespace separates sections from surrounding content.

## Attributes

| Attribute | Values | Description |
|---|---|---|
| `data-section` | (present) | Activates section layout. Required |
| `data-spacing` | `xs · sm · md · lg · xl` | External vertical margin (margin-block) |
| `data-padding` | `xs · sm · md · lg · xl` | Internal top/bottom padding |
| `data-size` | `sm · md · lg · xl · full` | Max-width constraint — centered inline |
| `data-bg` | `surface · raised · primary · muted · gradient` | Background color variant |
