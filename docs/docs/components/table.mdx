## Attributes

| Attribute | Type | Description |
|---|---|---|
| `data-sortable` | `boolean` | Enables sortable behavior |
| `aria-sort` | `'ascending' \| 'descending' \| 'none'` | Sort direction state |
| `is="ix-table"` | `string` | Enables sortable table enhancement |
