# Installation

Get started with Ignix Lite in a few simple steps.

Ignix Lite is a lightweight semantic UI toolkit built with
native HTML, CSS variables.

---

# Install Package

```bash
# npm
npm install @mindfiredigital/ignix-lite

```

---


# Basic Usage

Use semantic HTML with Ignix Lite styles.

```html
<button data-intent="primary">
  Primary Button
</button>

<button data-intent="danger">
  Delete
</button>

<input
  type="text"
  placeholder="Enter text"
/>
```

---

# Features

- Lightweight bundle size
- Semantic HTML first
- CSS variable powered
- Accessible components
- Dark mode support
- Minimal setup
- Tiny runtime overhead
