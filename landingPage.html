FAQ

Got Questions?

Is Ignix Lite really free?

Yes - the core library is MIT licensed and will always be free. Pro and Enterprise plans unlock premium components and support.

Do I need a build step?

Nope. Just include the CSS and JS files via CDN or npm and you're good to go.

How do I customise the design tokens?

Override CSS variables in your root stylesheet. Every color, radius, shadow and spacing value is a variable.
